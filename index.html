Microsoft API Vision vs Reddit Conversation Interpretation

This program scrapes various high scoring images from Reddit and feeds them into Microsoft's Vision API. We take various factors from this in order to determine a similarity rating between images. These images are all placed into a weighted graph.

At the same time, we will take the Reddit conversation scrape comments from the corresponding thread from which we got the image. We analyze the text to see if there are similarities among threads with images.

Is there a correlation between what Microsoft's Vision API can see as a similar image and what Reddit conversation can see as a similar image?

This type of information and correlation can be used to see if there is more to an image than what a machine can currently see. Images contain stories. At this time, no machine learning has been implemented. Using ML with this data can allow us to create "interpreted" understanding of an image rather than just what is explicitly in the image.
